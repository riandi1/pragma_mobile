# Prueba Técnica Flutter – Pragma

Este repositorio contiene la prueba técnica desarrollada en Flutter como parte del proceso de selección para Pragma. El objetivo principal fue consumir una API externa y mostrar la información de forma clara en una aplicación móvil funcional.

## Qué se solicitó

- Consumo de la API https://api.thecatapi.com/v1/breeds
- Uso de API Key
- Listado de razas de gatos
- Vista de detalle por raza
- Uso de componentes nativos
- Repositorio público

## Qué se implementó

- Consumo correcto de la API The Cat API
- Listado de razas con información relevante
- Navegación a una vista de detalle
- Manejo básico de estados (carga, error, datos)
- Interfaz simple y funcional en Flutter

## Cómo ejecutar el proyecto

### Clonar el repositorio
```
git clone https://github.com/tu-usuario/pragma-flutter-test.git
```

### Instalar dependencias
```
flutter pub get
```

### Ejecutar la app
```
flutter run
```

## Consideraciones técnicas

El enfoque fue priorizar la entrega funcional y el cumplimiento de los requisitos.  
El proyecto puede mejorarse aplicando patrones de diseño más robustos como Clean Architecture o MVVM, así como un manejo de estado más escalable y pruebas automatizadas.
